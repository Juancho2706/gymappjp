# GRO-002 — Vídeo demo (~2 min)

Checklist de producción (fuera del repo):

1. **Guión:** registro coach → alta alumno → asignar rutina → alumno completa entreno → check-in con fotos.  
2. **Grabación:** 1080p, audio claro, sin datos personales reales (cuentas de prueba).  
3. **Export:** MP4 corto; si el peso es alto, subir a YouTube (no listado) o Vimeo.  
4. **Publicación:** copiar ID de vídeo (11 caracteres) o URL `watch?v=` / `youtu.be/`.  
5. **App:** definir `NEXT_PUBLIC_DEMO_VIDEO_URL` en Vercel (o `.env.local`) — ver `DemoVideoSection` en la landing.

El repositorio incluye la sección **GRO-003** en la home (`DemoVideoSection` + CTA a registro/pricing) enlazada a esa variable de entorno.

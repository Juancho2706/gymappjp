— Principios

// 01

Mono para metadata. Rutas, estados, IDs, códigos. El mono es el idioma del sistema; el sans es el idioma del humano.

// 02

Shadow como gesto. Un solo box-shadow 8px 8px 0 en negro puro. Una sola vez (el phone-frame). Nunca más.

// 03

Sin drag-and-drop en móvil. El problema de gestos se resuelve con el patrón FAB + Bottom Sheet. El coach toca "+", elige, vuelve.
